#CONVERSOR DE MOEDAS EM JAVA

Codigo feito para treinar Java em um projeto "real" aonde criei esse conversor de moedas, aonde recebo os inputs de qual moeda a sera convertida
, para qual moeda converter e qual valor a ser convertido. 

Importante informar que os valores da moedas estão indexados ao BRL, onde se converte primeiro para real e depois e feita as outras conversões.
Os valores das moedas são fixas, e alteradas manualmente na criação do HashMap para cada objeto da classe Currency (moeda)
